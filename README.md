# توثيق العمليات التشغيلية — السعودية للطاقة

نظام توثيق العمليات التشغيلية الميدانية لإدارة كهرباء الخرج  
دائرة دعم العمليات والصيانة

---

## المميزات

- **توثيق العمليات** — نوع العملية / المشغل / الدائرة / رقم المهمة / نوع المفتاح
- **GPS تلقائي** — يلتقط الموقع عند فتح النموذج
- **الصور** — التقاط صورة المعدة مباشرة من الكاميرا
- **لوحة إدارة** محمية برقم سري (SHA-256)
- **رفع بالجملة** — أسماء الفنيين والمهام دفعة واحدة
- **تقرير يومي** قابل للطباعة — حسب الموظف والدائرة ونسبة الالتزام
- **تصدير CSV** يفتح مباشرة في Excel
- **يعمل بدون إنترنت** — يحفظ محلياً ويزامن عند الاتصال
